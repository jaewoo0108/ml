---

## 1. 지도학습 (Supervised Learning)

정답(label)이 있는 데이터를 모델에게 제공해 학습.

지도학습의 유형은 분류와 회귀로 나뉜다.

1. 분류: 유한한 집합의 카데고리, 개별 범주를 예측
2. 회귀: 무한한 집합에서 숫자값을 예측

---

## 2. 비지도학습 (Unsupervised Learning)

정답(label)이 없는 데이터를 모델에게 제공해 정답을 예측하는게 아닌 데이터셋에서 구조적인 패턴을 찾거나 흥미로운 것을 찾는 것.

비지도학습의 유형은 클러스터링, 이상 탐지(anomaly detection), 차원 축소로 나뉜다.

1. 클러스터링: 데이터를 서로 다른 그룹 또는 군집으로 할당
2. 이상치 탐지: 비정상적인 이벤트 탐지하는데 사용
3. 차원 축소: 큰 데이터 집합을 가져와서 정보를 최대한 적게 손실하면서 더 작은 크기의 데이터 집합으로 압축하는 것

---

## 3. 선형 회귀 (Linear Regression)

- 훈련세트: 모델을 훈련하기 위한 데이터
- 일반적인 머신 러닝에서 x를 입력값, f(x)를 모델이라 할때 y hat을 모델의 출력값으로 표현한다.
- 선형함수는 직선에 대한 다른 용어. 결국 선형회귀라는건 데이터셋에 가장 잘 맞는 직선을 설정하는것.

$$f_{w,b}(x) = wx + b$$

- `f(x) = wx + b`인 경우에 w, b는 모델의 매개변수(가중치)이며 모델을 개선하기 위해 훈련중에 조정하는 변수
- 매개변수 w와 b를 설정해 데이터셋에 가장 잘 맞는 직선 설정

---

## 4. 비용 함수 (Cost Function)

- 모델의 예측값과 실제 참 값의 차이를 측정하는 함수
- (모델의 예측값 - 참값)의 제곱의 평균을 구한다

선형 회귀의 최종 목표: 비용함수의 출력값이 가장 최소화되는 모델의 매개변수를 찾아서 설정하는것.
