## 1. 특성 스케일링 (Feature Scaling)

각각의 입력 데이터들에서 각 특성이 가질 수 있는 값들의 범위가 큰 경우엔 해당 특성의 파라미터 값을 상대적으로 작은 값으로 설정하고, 값들의 범위가 작은 경우엔 상대적으로 큰 값으로 설정한다.
(= 특성 값의 범위를 비슷한 수준으로 가지게 하는것)

### 여러 종류의 특성 스케일링

- Min-Max Normalization (최소-최대 정규화): x max - x min 분의 x - x min

$$x' = \frac{x - x_{min}}{x_{max} - x_{min}}$$

- Mean normalization (평균 정규화): 각 데이터에서 평균 값을 빼고, 데이터의 범위로 나눠주는 기법
- Z-score normalization: 각 데이터에서 평균 값을 빼고, 데이터의 범위 대신 표준편차(데이터가 중심에서 얼마나 흩어져 있는 정도)로 나눠주는 기법

평균정규화, 최소-최대 정규화에선 이상치가 하나라도 껴있으면 전체 데이터의 범위가 망가지지만 z-점수 정규화는 표준편차를 사용하기에 안정적이다.

훈련데이터(입력값)는 정규화 과정을 거치지만 실제 정답 데이터는 정규화를 하지 않아도 된다. 입력데이터가 정규화되어 스케일이 작아지더라도 모델의 가중치와 편향이 정답 스케일에 대응해 필요한만큼 커질 수 있기에 실제 정답값을 정확히 예측 가능.

## 2. 경사하강법이 잘 작동하고 있는지 확인하는 방법

경사 하강법의 목표: 비용함수 값을 최소화하는 파라미터 w, b를 찾는 것.

y축을 비용함수의 값으로, x축을 경사하강법의 iteration(반복) 횟수로 설정하고 그린 그래프를 learning curve라 한다.

- learning curve를 봄으로써 경사하강법이 수렴하는 곳을 확인 가능
- 경사하강법이 잘 작동한다는 것은 학습률이 잘 설정되어 있다면, 비용함수값이 매 반복마다(every single iteration) 줄어든다는 것이다

learning curve 그래프에서 iterations 횟수가 증가함에도 비용함수 값이 증가하는 그래프가 그려진다면 →

1. 학습률이 너무 크거나
2. 코드에 버그가 있다

→ 경사하강법을 수행한다면 학습률 값으로 여러 범위의 값을 시도해봄으로써 다양한 learning curve를 보고 적합한 학습률 값을 고를 수 있다.

### 수렴 (Converge)

학습을 계속 반복해도 파라미터 값이 더 이상 유의미하게 변하지 않고 특정값에 머무르는것 (= 모델 학습을 완료했다).

수렴점은 비용함수가 완벽히 0값이 되는 지점이 아니다. 계속 줄어들 여지가 남아있음에도 불구하고 학습을 멈추는 것 → 과적합 방지, 자원의 낭비, 부동 소수점 한계로 인해.

## 3. 다중회귀에서의 경사하강법

$$
\begin{aligned}
&\text{repeat until convergence} \lbrace \\
&\quad w_j := w_j - \alpha \frac{\partial J(\mathbf{w}, b)}{\partial w_j} \quad \text{for } j = 0..n-1 \\
&\quad b := b - \alpha \frac{\partial J(\mathbf{w}, b)}{\partial b} \\
&\rbrace
\end{aligned}
$$

여기서 각 기울기는 다음과 같이 계산된다:

$$\frac{\partial J(\mathbf{w}, b)}{\partial w_j} = \frac{1}{m} \sum_{i=0}^{m-1} (f_{\mathbf{w},b}(\mathbf{x}^{(i)}) - y^{(i)}) x_j^{(i)}$$

$$\frac{\partial J(\mathbf{w}, b)}{\partial b} = \frac{1}{m} \sum_{i=0}^{m-1} (f_{\mathbf{w},b}(\mathbf{x}^{(i)}) - y^{(i)})$$

- 하나의 특정 가중치 w j를 미세하게 움직일때 전체 비용함수의 값은 얼마나 변하는가를 나타낸다.
- 전체 데이터셋에 대해 가중치의 기울기를 각각 계산한 뒤 그 평균값을 구하고, 여기에 매 단계마다 학습률을 곱해서 가중치를 업데이트한다.
- 각 가중치에 대해서 경사하강법을 수행할 때, 서로 다른 데이터의 크기 때문에 특정 가중치는 한번에 엄청나게 큰 크기로 업데이트 되는 반면 작은 데이터값을 가지는 가중치는 아주 미세하게 업데이트 되어버리기에 학습이 끝날 때까지 오차가 더 이상 줄어들지 않고 정체되는 현상 발생.
